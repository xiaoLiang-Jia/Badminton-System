## 一、普通用户操作手册

### 1.1 账号注册与登录

| 功能 | 接口路径 | 说明 |
|-----|---------|------|
| 注册 | `POST /api/user/register` | 输入用户名、密码、真实姓名、手机号、邮箱 |
| 登录 | `POST /api/user/login` | 输入用户名和密码，返回JWT Token |
| 修改密码 | `POST /api/user/changePassword` | 需认证后操作 |

### 1.2 场地预约

| 功能 | 接口路径 | 说明 |
|-----|---------|------|
| 查看场地 | `GET /api/court/list` | 公开接口，查看所有场地 |
| 查看场地详情 | `GET /api/court/{id}` | 查看特定场地信息 |
| 查看可用场地 | `GET /api/court/available` | 查看可选日期的可用场地 |
| 创建预约 | `POST /api/booking/create` | 选择场地、日期、时间段创建预约 |
| 查看我的预约 | `GET /api/booking/list` | 查看个人所有预约记录 |
| 取消预约 | `POST /api/booking/cancel/{id}` | 取消未支付的预约 |
| 支付预约 | `POST /api/booking/pay/{id}` | 使用余额支付预约 |

**预约流程：**
1. 进入场地列表页 → 选择场地
2. 选择预约日期和时间段
3. 确认订单信息 → 提交预约
4. 选择支付方式（余额支付）
5. 支付成功后收到预约成功通知

### 1.3 课程报名

| 功能 | 接口路径 | 说明 |
|-----|---------|------|
| 查看课程 | `GET /api/course/list` | 查看所有可报名课程 |
| 查看课程详情 | `GET /api/course/{id}` | 查看课程详细信息 |
| 报名课程 | `POST /api/course/enroll` | 报名参加课程 |
| 取消报名 | `POST /api/course/cancel/{id}` | 取消课程报名 |
| 查看我的课程 | `GET /api/course/my-courses` | 查看已报名的课程 |

### 1.4 论坛社区

| 功能 | 接口路径 | 说明 |
|-----|---------|------|
| 查看帖子 | `GET /api/forum/list` | 浏览帖子列表 |
| 查看帖子详情 | `GET /api/forum/{id}` | 查看帖子内容和评论 |
| 发布帖子 | `POST /api/forum/publish` | 发布新帖子 |
| 更新帖子 | `PUT /api/forum/update` | 修改自己发布的帖子 |
| 删除帖子 | `DELETE /api/forum/delete/{id}` | 删除自己发布的帖子 |
| 点赞帖子 | `POST /api/forum/like/{id}` | 为帖子点赞 |
| 发表评论 | `POST /api/forum/comment` | 在帖子下发表评论 |
| 查看评论 | `GET /api/forum/comments/{id}` | 查看帖子评论列表 |

**帖子分类：**
- 1-技术讨论
- 2-约球
- 3-比赛心得
- 4-器材交流
- 5-新手求助

### 1.5 个人中心

| 功能 | 接口路径 | 说明 |
|-----|---------|------|
| 查看个人信息 | `GET /api/user/info` | 获取当前用户信息 |
| 更新个人信息 | `PUT /api/user/update` | 修改个人资料 |
| 查看余额 | `GET /api/user/balance` | 查看账户余额 |
| 充值 | `POST /api/user/recharge` | 账户充值 |

## 二、教练操作手册

教练用户包含普通用户的所有功能，同时额外拥有以下权限：

### 2.1 课程管理

| 功能 | 接口路径 | 说明 | 权限 |
|-----|---------|------|------|
| 创建课程 | `POST /api/course/add` | 创建新课程 | 教练/管理员 |
| 更新课程 | `PUT /api/course/update` | 修改课程信息 | 教练/管理员 |
| 删除课程 | `DELETE /api/course/delete/{id}` | 删除课程 | 教练/管理员 |

### 2.2 课程设置内容

创建课程时可设置以下信息：
- 课程名称 (`name`)
- 课程描述 (`description`)
- 价格 (`price`)
- 最大学员数 (`max_students`)
- 课时时长 (`duration`)
- 课程安排 (`schedule`)
- 难度等级 (`level`): 1-初级 2-中级 3-高级
- 上课场地 (`court_id`)
- 课程图片 (`image_url`)

### 2.3 学员管理

- 查看已报名学员列表
- 管理系统学员的考勤状态

## 四、通用操作流程

### 4.1 登录流程

```
1. 打开登录页 /login
2. 输入用户名和密码
3. 点击登录
4. 后端验证并返回JWT Token
5. 前端存储Token到LocalStorage
6. 跳转到首页
```

### 4.2 预约支付流程

```
1. 选择场地
2. 选择日期和时间段
3. 确认订单（系统创建待支付预约）
4. 点击支付
5. 扣除用户余额
6. 更新预约状态为已支付
7. 显示预约成功
```

### 4.3 课程报名流程

```
1. 浏览课程列表
2. 选择课程查看详情
3. 点击报名
4. 确认报名信息
5. 使用余额支付
6. 更新课程报名信息
7. 显示报名成功
```

## 六、权限说明

- **公开接口**：无需登录即可访问（如登录、注册、场地列表、课程列表）
- **需认证接口**：登录后即可访问（如预约、课程报名、论坛发帖）
- **管理员接口**：仅管理员可访问（如用户管理、场地管理、���据��计）
- **教练接口**：教练角色可访问（如课程管理）

## 七、注意事项

1. 预约场地前需确保账户余额充足
2. 课程报名后如需取消，请在开课前操作
3. 发布论坛帖子需遵守社区规范，违规内容将被删除
4. 管理员操作需谨慎，删除操作不可恢复
5. 教练创建课程时需设置合理的上课时间和人数
